# Довідник флори та фауни

Індивідуальний проєкт із дисципліни "Веб-технології". Це веб-додаток для ознайомлення з біорізноманіттям (рослинний та тваринний світ), розроблений для коледжу.

## Про проєкт

Проєкт реалізовано на чистому **PHP** без використання сторонніх фреймворків (Laravel, Symfony тощо), за вимогою викладача. Для стилізації використано **Bootstrap 5** у мінімалістичному стилі.

### Основні функції:
- **Публічний каталог**: перегляд списку рослин та тварин.
- **Фільтрація**: сортування записів за категоріями (Флора/Фауна).
- **Детальні сторінки**: перегляд повної інформації про конкретний вид.
- **Адмін-панель**: захищений інтерфейс для керування контентом.
- **CRUD**: можливість додавання, редагування та видалення записів разом із зображеннями.

## Архітектура та реалізація

Проєкт побудований на архітектурному шаблоні **Micro-MVC**:
- **Core (Init/Functions)**: ядро системи, що відповідає за підключення до БД, завантаження конфігурації `.env` та загальні функції.
- **Controllers**: окремі файли в корені та папці `admin/`, які обробляють логіку запитів.
- **Views**: шаблони відображення, винесені в окрему директорію для чистоти коду.

### Безпека:
- **PDO Prepared Statements**: захист від SQL-ін'єкцій.
- **CSRF Protection**: захист форм в адмін-панелі від міжсайтової підробки запитів.
- **Bcrypt Hashing**: безпечне зберігання паролів адміністраторів.
- **Environment variables**: збереження чутливих даних (БД) у файлі `.env`.

## Технологічний стек
- **Backend**: PHP 8.3+
- **Database**: MySQL (InnoDB)
- **Frontend**: HTML5, CSS3, Bootstrap 5
- **Dependencies**: PHP Dotenv (через Composer)

## Встановлення та запуск

1.  Клонуйте репозиторій.
2.  Встановіть залежності:
    ```bash
    composer install
    ```
3.  Налаштуйте з'єднання з базою даних у файлі `.env`.
4.  Імпортуйте структуру БД:
    ```bash
    mysql -u [user] -p [db_name] < database.sql
    ```
5.  Запустіть вбудований сервер:
    ```bash
    php -S localhost:8000
    ```
