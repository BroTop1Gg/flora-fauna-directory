# Довідник Флори та Фауни

Цей проєкт є навчальним індивідуальним завданням з веб-програмування. Він являє собою каталог видів флори та фауни з можливістю адміністрування.

## Особливості проєкту

- **Zero Dependency**: Проєкт повністю написаний на чистому PHP (Vanilla PHP) без сторонніх бібліотек. Це забезпечує максимальну сумісність із будь-яким хостингом (зокрема InfinityFree).
- **Адаптивний дизайн**: Використано Bootstrap 5 для зручного відображення на мобільних пристроях.
- **Безпека**: Власна реалізація завантаження середовища (`.env`), CSRF-захист та безпечне підключення до БД через PDO.

## Вимоги

- PHP 7.4 або вище
- MySQL / MariaDB
- FTP-клієнт (наприклад, FileZilla) для деплою

## Встановлення та деплой

### 1. Налаштування Бази Даних
1. Імпортуйте файл `database.sql` у вашу базу даних MySQL.
   - *Примітка*: Якщо ви оновлюєте існуючий проєкт, спочатку видаліть старі таблиці.

### 2. Конфігурація
1. Відредагуйте файл `.env` у корені проєкту, вказавши ваші параметри підключення до БД:
   ```env
   DB_HOST=ваш_хост
   DB_NAME=назва_бд
   DB_USER=користувач
   DB_PASS=пароль
   ```

### 3. Деплой на хостинг
1. Використовуйте **FileZilla** для підключення до вашого хостингу.
2. Завантажте весь вміст папки проєкту в директорію `htdocs` (або `public_html`) на сервері.
3. Переконайтеся, що файл `.htaccess` у папці `img/` завантажений коректно — він захищає ваші зображення.

## Адміністрування
Панель адміністратора доступна за адресою `/admin`.
Стандартні дані (після імпорту `database.sql`):
- **Логін**: `admin`
- **Пароль**: `admin123` (або як налаштовано в БД)
